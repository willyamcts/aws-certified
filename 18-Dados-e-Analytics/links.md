## Comparativo de Serviços Analytics

| Serviço | Caso de Uso | Latência |
|---------|------------|----------|
| Athena | SQL ad-hoc em S3 | Segundos |
| Redshift | Data warehouse OLAP | Segundos |
| OpenSearch | Busca full-text + logs | Milissegundos |
| QuickSight | Dashboards BI | Segundos |
| Kinesis Data Streams | Streaming em tempo real | Milissegundos |
| Kinesis Firehose | ETL stream para storage | Segundos-minutos |
