## Os 6 Pilares — Palavras-Chave para o Exame

| Pilar | Palavras-Chave |
|-------|---------------|
| Operational Excellence | IaC, runbooks, eventos, melhoria contínua |
| Security | least privilege, MFA, encryption at-rest/in-transit, detective controls |
| Reliability | health checks, backups, multi-AZ, chaos engineering, auto recovery |
| Performance Efficiency | right-sizing, caching, CDN, global reach |
| Cost Optimization | Reserved Instances, Spot, right-sizing, lifecycle, delete unused |
| Sustainability | eficiência de hardware, minimizar footprint, região com energia renovável |
